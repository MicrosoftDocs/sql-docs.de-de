**Zusätzliche Leerzeichen**

Führende und nachfolgende Leerzeichen werden ignoriert, wenn SQL-Zeichendaten in einen der folgenden Typen konvertiert werden:

- date
- NUMERIC
- time
- timestamp
- Intervall-C-Daten
