# <a name="sqlbulkoperations-and-the-cursor-library"></a>SQLBulkOperations und die Cursorbibliothek
> [!IMPORTANT]  
>  Diese Funktion wird in einer zukünftigen Version von Windows entfernt. Vermeiden Sie die Verwendung dieses Features bei der Entwicklung neuer Anwendungen, und planen Sie das Ändern von Anwendungen, in denen diese Funktion derzeit verwendet wird Microsoft empfiehlt die Verwendung der Cursor-Funktionalität des Treibers.  
  
 Das Aufrufen von **SQLBulkOperations** wird von der Cursor Bibliothek nicht unterstützt. Allgemeine Informationen zu **SQLBulkOperations** finden Sie unter [SQLBulkOperations-Funktion](../../../odbc/reference/syntax/sqlbulkoperations-function.md).
