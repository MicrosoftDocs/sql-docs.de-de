# <a name="scalar-functions"></a>Skalarfunktionen
Die Microsoft ODBC Desktop-Datenbanktreiber unterstützen die folgenden Skalarfunktionen:  

:::row:::
    :::column:::
        CONCAT  
        CONVERT  
        CURDATE  
        CURTIME  
        DATABASE  
        DAYOFMONTH  
        TAGDERWOCHE  
    :::column-end:::
    :::column:::
        LCASE  
        LEFT  
        LENGTH  
        LOCATE  
        LTRIM  
        MOD  
        MONTH  
    :::column-end:::
    :::column:::
        RIGHT  
        RTRIM  
        SUBSTRING  
        UCASE  
        USER  
        YEAR  
    :::column-end:::
:::row-end:::

 Informationen zu den Argumenten und Rückgabe Werten von skalaren Funktionen finden Sie in Anhang F der *ODBC-Programmier Referenz.*
