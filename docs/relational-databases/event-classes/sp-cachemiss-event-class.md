# <a name="spcachemiss-event-class"></a>SP:CacheMiss-Ereignisklasse
[!INCLUDE [SQL Server - ASDB](../../includes/applies-to-version/sql-asdb.md)]
  Von der SP:CacheMiss-Ereignisklasse wird angegeben, dass die Prozedur nicht im Cache gefunden wurde. Wenn die Ereignisklasse „SP:CacheMiss“ häufig auftritt, kann das ein Hinweis darauf sein, dass für [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] mehr Arbeitsspeicher bereitgestellt werden sollte, um die Größe des Prozedurcaches zu erhöhen.
