# <a name="ibcpsession2-native-client-ole-db-provider"></a>IBCPSession2 (Native Client OLE DB-Anbieter)
[!INCLUDE[SQL Server Azure SQL Database Synapse Analytics PDW ](../../includes/applies-to-version/sql-asdb-asdbmi-asa-pdw.md)]

  Die IBCPSession2-Schnittstelle ist eine Erweiterung von IBCPSession, die eine Elementfunktion bereitstellt. Diese stellt eine Alternative zum Aufrufen von IBCPSession:: BCPColFmt für jede Spalte dar.  IBCPSession2 erbt von IBCPSession und fügt eine neue Methode hinzu: [IBCPSession2::BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
