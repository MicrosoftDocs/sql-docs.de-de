# <a name="activeconnection-property-adox"></a>ActiveConnection-Eigenschaft (ADOX)
Gibt das ADO- [Verbindungs](../ado-api/connection-object-ado.md) Objekt an, zu dem der [Katalog](./catalog-object-adox.md) gehört.  
  
## <a name="settings-and-return-values"></a>Einstellungen und Rückgabewerte  
 Legt ein **Verbindungs** Objekt oder eine **Zeichenfolge** fest, die die Definition für eine Verbindung enthält. Gibt das aktive **Verbindungs** Objekt zurück.  
  
## <a name="remarks"></a>Bemerkungen  
 Der Standardwert ist ein NULL-Objekt Verweis.  
  
## <a name="applies-to"></a>Gilt für  
 [Catalog-Objekt (ADOX)](./catalog-object-adox.md)
